[cite_start]**Focus Through Fun** is an innovative, web-based serious game called **Nimrod** designed to assist individuals with Attention Deficit Hyperactivity Disorder (ADHD)[cite: 9, 25]. [cite_start]By combining engaging gameplay mechanics with interactive cognitive training, the application aims to help users develop, improve, and maintain focus, attentional control, and task persistence[cite: 8, 25, 27].

[cite_start]Traditional ADHD treatment methods often face challenges like low adherence rates due to side effects or a lack of user engagement[cite: 7, 13]. [cite_start]Nimrod addresses this gap by offering an enjoyable, sustainable, and accessible digital complement to traditional therapies[cite: 10, 14, 16].

---

## 🚀 Key Features

* [cite_start]**Functional Game Framework:** Features a core "clicker" mechanic built seamlessly for immediate audio-visual feedback and interaction[cite: 40, 41, 140].
* [cite_start]**Dynamic Difficulty Scaling:** Implements a sophisticated progression system where monster health scales up after defeating bosses, ensuring a continuous cognitive challenge while preventing boredom[cite: 18, 28, 137].
* [cite_start]**Strategic Upgrade Shop:** An in-game marketplace where players spend currency earned from gameplay to enhance capabilities like tap damage, triple strike, and adrenaline rush[cite: 51, 52, 137].
* [cite_start]**Avatar Customization:** Features a cosmetic system where users can purchase and personalize character skins using coins earned from boss battles[cite: 21, 67, 68].
* [cite_start]**Real-Time Performance Tracking:** Monitors player metrics—including Total Clicks, Monsters Slain, Total Coins Earned, and Critical Hit %[cite: 130].
* [cite_start]**Achievements & Goals System:** Provides explicit objectives to strive for, offering tangible feedback on progress to drive positive reinforcement and motivation[cite: 30, 131].
* [cite_start]**Cross-Platform Accessibility:** Optimized to run seamlessly directly in desktop web browsers or on Android devices via local HTML file execution[cite: 33, 34, 37].
* [cite_start]**Session Persistence:** Integrated local save game functionality allows users to store their progress and resume their cognitive training at any time[cite: 133, 140].

---

## 🛠️ Tech Stack

[cite_start]Nimrod is built on a lightweight, dependency-free front-end architecture to ensure smooth performance across various hardware profiles[cite: 37, 139]:

* [cite_start]**HTML5:** Handles the semantic structure and structural layout of the game environment[cite: 35].
* [cite_start]**CSS3:** Delivers visually appealing, attractive designs to maximize user engagement[cite: 35].
* [cite_start]**JavaScript (ES6):** Powers the core game loop, dynamic shop mathematics, audio triggers, UI interactivity, and data tracking[cite: 35, 36].
