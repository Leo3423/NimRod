## About The Project

[cite_start]**Focus Through Fun: Nimrod** is an innovative, web-based serious game designed to support individuals with Attention Deficit Hyperactivity Disorder (ADHD)[cite: 211, 214]. [cite_start]By combining engaging gameplay mechanics with interactive cognitive training, the application aims to improve focus, concentration, attentional control, and task persistence[cite: 213, 214, 230]. 

[cite_start]Traditional ADHD treatments often suffer from low engagement and adherence rates[cite: 212, 218]. [cite_start]Nimrod addresses this gap by offering an accessible, digital alternative or complement to conventional therapies [cite: 215, 221][cite_start], leveraging research that shows interactive video games can positively enhance cognitive functions and executive processing speed[cite: 213, 220].

### Key Features

* [cite_start]**Dynamic Cognitive Stimulation:** Employs a progressive difficulty system where enemy scaling ensures continuous cognitive engagement and prevents boredom or frustration[cite: 223, 233, 342].
* [cite_start]**Strategic Upgrade Shop:** Features an in-game marketplace where players can spend currency earned through gameplay to upgrade combat capabilities (e.g., tap count, triple strike, adrenaline rush)[cite: 225, 257].
* [cite_start]**Avatar Customization:** Allows players to unlock and personalize character skins using rewards earned from defeating boss encounters, fostering long-term motivation[cite: 226, 235, 273].
* [cite_start]**Statistics & Achievement Tracking:** Integrates real-time performance tracking—including total clicks, monsters slain, and critical hit percentages—providing tangible feedback on player progress[cite: 235, 335, 336].
* [cite_start]**Immersive Audio-Visual Feedback:** Built with scene-specific background art, background music, and real-time interaction feedback to maintain sensory immersion[cite: 241, 289, 345].
* [cite_start]**Cross-Platform Accessibility:** Features a light web-framework architecture that can be seamlessly played on both desktop web browsers and Android devices via local HTML execution[cite: 238, 239, 242].
* [cite_start]**Session Persistence:** Includes built-in save game functionality, allowing users to store progress and resume cognitive training at their convenience[cite: 338].

### Built With

[cite_start]The application is built entirely on a lightweight, smooth front-end stack[cite: 242, 344]:

* [cite_start]**HTML5** – Game structure and foundational semantic layout[cite: 240].
* [cite_start]**CSS3** – Visual art design, dynamic layout styling, and themes[cite: 240].
* [cite_start]**JavaScript (ES6)** – Core game mechanics, interactivity engine, real-time math trackers, and audio triggers[cite: 240, 241].
