# Focus Through Fun: Nimrod

[![Tech Stack](https://img.shields.io/badge/Stack-HTML5%20%7C%20CSS3%20%7C%20JavaScript-blue)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![Platform](https://img.shields.io/badge/Platform-Web%20%7C%20Android-green)](https://developer.android.com/)

 **Focus Through Fun** is an innovative  web-based serious game called **Nimrod** designed to assist individuals with Attention Deficit Hyperactivity Disorder (ADHD).  By combining engaging gameplay mechanics with interactive cognitive training  the application aims to help users develop  improve  and maintain focus  attentional control  and task persistence.

 Traditional ADHD treatment methods often face challenges like low adherence rates due to side effects or a lack of user engagement.  Nimrod addresses this gap by offering an enjoyable  sustainable  and accessible digital complement to traditional therapies.

---

## 🚀 Key Features

*  **Functional Game Framework:** Features a core "clicker" mechanic built seamlessly for immediate audio-visual feedback and interaction.
*  **Dynamic Difficulty Scaling:** Implements a sophisticated progression system where monster health scales up after defeating bosses  ensuring a continuous cognitive challenge while preventing boredom.
*  **Strategic Upgrade Shop:** An in-game marketplace where players spend currency earned from gameplay to enhance capabilities like tap damage  triple strike  and adrenaline rush .
*  **Avatar Customization:** Features a cosmetic system where users can purchase and personalize character skins using coins earned from boss battles .
*  **Real-Time Performance Tracking:** Monitors player metrics—including Total Clicks  Monsters Slain  Total Coins Earned  and Critical Hit.
*  **Achievements & Goals System:** Provides explicit objectives to strive for  offering tangible feedback on progress to drive positive reinforcement and motivation .
*  **Cross-Platform Accessibility:** Optimized to run seamlessly directly in desktop web browsers or on Android devices via local HTML file execution .
*  **Session Persistence:** Integrated local save game functionality allows users to store their progress and resume their cognitive training at any time .

---

## 🛠️ Tech Stack

 Nimrod is built on a lightweight  dependency-free front-end architecture to ensure smooth performance across various hardware profiles :

*  **HTML :** Handles the semantic structure and structural layout of the game environment.
*  **CSS :** Delivers visually appealing  attractive designs to maximize user engagement.
*  **JavaScript (ES ):** Powers the core game loop  dynamic shop mathematics  audio triggers  UI interactivity  and data tracking .
